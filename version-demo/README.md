# Version Timeline Demo

基于 React + shadcn/ui + Tailwind CSS 的版本管理时间线组件

## 技术栈

- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式系统
- **shadcn/ui 设计理念** - 组件设计
- **Lucide React** - 图标库
- **Vite** - 构建工具

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 预览

```bash
npm run preview
```

## 设计规范

遵循项目的 UIAgentRules.md 规范：
- ✅ 使用 Tailwind CSS 语义化颜色
- ✅ 使用 Lucide React 图标
- ✅ 遵循 CRAP 设计原则
- ✅ 微交互动效（150-300ms）
- ✅ 深色模式支持
