# 🔧 线段颜色和位置修复

## 修复内容

### 1. 线段颜色调整 ✅

#### 主版本线段
- **颜色**: `#3b82f6` (蓝色)
- **应用位置**:
  - 中心垂直主轴
  - 时间线节点（方块）
  - 横向连接线

#### 子版本线段
- **颜色**: `#67e8f9` (青色)
- **应用位置**:
  - 左侧垂直分支线
  - 子版本圆点
  - 子版本标题文字
  - "Add Sub" 按钮边框

### 2. 横向线段位置修复 ✅

**修复前**:
- 所有横向线段都从节点向右延伸

**修复后**:
- 卡片在**左侧**时 → 线段从节点向**左**延伸
- 卡片在**右侧**时 → 线段从节点向**右**延伸

#### 实现细节

```tsx
// TimelineNode.tsx
<div
  className={cn(
    'absolute h-1 bg-timeline-main',
    position === 'left' 
      ? 'right-1/2 left-[-2rem] w-[calc(2rem+0.5px)]'  // 向左延伸
      : 'left-1/2 right-[-2rem] w-[calc(2rem+0.5px)]',  // 向右延伸
  )}
/>
```

### 3. 自定义颜色配置

在 `tailwind.config.js` 中添加:

```javascript
colors: {
  timeline: {
    main: "#3b82f6",  // 蓝色 - 主版本
    sub: "#67e8f9",   // 青色 - 子版本
  },
}
```
