# EchoAIUIDemo 

EchoAIUIDemo 是一个用于展示 EchoAI 产品设计和交互效果的演示项目。通过这个项目，你可以直观地体验 EchoAI 的功能和界面设计。以提供给具体的项目的ui实现作为参考。

## fab-demo
一个基于 React + Tailwind CSS 的浮动操作按钮（FAB）交互演示项目，展示了现代化的多层级面板展开效果和流畅的动画交互体验。
### 核心特性- 🎯 **渐进式交互设计**：FAB → 列表面板 → 编辑面板，三层递进式展开
- ⚡ **高性能动画**：所有动画控制在 150-180ms，确保流畅体验- 🎨 **shadcn/ui 设计系统**：完全遵循现代化设计规范和配色体系
- 📱 **响应式布局**：支持桌面端和移动端，FAB 位置可左右切换- 🧩 **组件化架构**：10+ 个独立组件，代码结构清晰易维护
### 适用场景
- AI 对话应用的提示词快捷访问- 移动端应用的快捷操作入口
- 多层级内容管理界面- 需要优雅展开/收起动画的场景
